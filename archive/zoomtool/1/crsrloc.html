crsrloc

(./crsrloc.m)

Function Synopsis

[y] = crsrloc(ax,name)

Help text

CRSRLOC Return axes cursor position.
       y=crsrloc(axes,'name') returns the x- or y-axes cursor
       location dependent upon if the cursor is a vertical or
       horizontal cursor.

       See also CRSRCR, CRSRDEL, CRSRON, CRSROFF
