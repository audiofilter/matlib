 Object count/get vector of handles functions.
   CNTAXES     - Count axes objects.
   CNTLINES    - Count line axes objects.
   CNTUICTL    - Count figure uicontrols objects.

 Axes cursor functions
   CRSRDEL     - Delete axes cursor.
   CRSRLOC     - Return axes cursor position.
   CRSRMAKE    - Create axes crsr.
   CRSRMOVE    - Move axes cursor.
   CRSROFF     - Turn axes crsr off.
   CRSRON      - Turn axes cursor on.

 Figure window object find functions
   FINDAXES    - Find axes object.
   FINDCHKB    - Find checkbox uicontrol.
   FINDEDIT    - Find edit uicontrol.
   FINDFRAM    - Find frame uicontrol.
   FINDMENU    - Find uimenu uicontrol.
   FINDPOPU    - Find popupmenu uicontrol.
   FINDPUSH    - Find pushbutton uicontrol.
   FINDRDIO    - Find radiobutton uicontrol.
   FINDSLID    - Find radiobutton uicontrol.
   FINDUITX    - Find text uicontrol.
   IDAXES      - Assign identifier to axes (see FINDAXES).

 Axes object find function
   FINDLINE    - Find axes line object.
   FINDTEXT    - Find axes text object.

 Popupmenu access functions
   GETPOPVL    - Return current numeric value on popup menus.
   GETPOPST    - Return current string label on popup menus.
   POPHITE     - Adjust height of popup menu for MS-Windows.

 Text scripts and functions
   PROGTEST    - Test file for programmer toolbox.
   TESTZOOM    - Test zoom controls.

   ZOOMLEFT    - Used by ZOOMTOOL to move cursors one sample to the
   ZOOMMOVE    - Used by ZOOMTOOL to move nearest cursor to mouse
   ZOOMPKLF    - Used by ZOOMTOOL to move cursors one extrema to the
   ZOOMPKRT    - Used by ZOOMTOOL to move cursors one extrema to the
   ZOOMRGHT    - Used by ZOOMTOOL to move cursors one sample to the
   ZOOMSET     - Used by ZOOMTOOL to move position cursors according to
   ZOOMTGGL    - Used by ZOOMTOOL to move toggle attachment of
   ZOOMTOOL    - Axes zoom/measurement tool.
   ZOOMUP      - Used by ZOOMTOOL to move nearest cursor to mouse
   ZOOMXFUL    - Used by ZOOMTOOL to zoom the X axis to full limits.
   ZOOMXIN     - Used by ZOOMTOOL to zoom the x axis in.
   ZOOMXOUT    - Used by ZOOMTOOL to zoom the x axis out.
   ZOOMYFUL    - Used by ZOOMTOOL to zoom the y axis to full limits.
   ZOOMYIN     - Used by ZOOMTOOL to zoom the y axis in.
   ZOOMYOUT    - Used by ZOOMTOOL to zoom the y axis out.
